====== Std ======
''Std'' provides the basic types and methods of Wrapl. Note that most methods provided for the basic types (''[[.Std:Integer#T|Integer.T]]'', ''[[.Std:String#T|String.T]]'', etc) are actually provided by the module ''[[.Std:Methods|Std.Methods]]'', and will documented in ''[[.Std:Methods|Std.Methods]]''.
