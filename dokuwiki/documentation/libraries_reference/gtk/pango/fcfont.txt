PangoFcFont is a base class for font implementation using the Fontconfig and
FreeType libraries. It is used in the Xft and FreeType backends shipped with
Pango, but can also be used when creating new backends. Any backend deriving
from this base class will take advantage of the wide range of shapers
implemented using FreeType that come with Pango.
