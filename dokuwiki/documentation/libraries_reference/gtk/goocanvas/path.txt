GooCanvasPath represents a path item, which is a series of one or more lines,
bezier curves, or elliptical arcs.
It is a subclass of GooCanvasItemSimple and so inherits all of the style
properties such as "stroke-color", "fill-color" and "line-width".
It also implements the GooCanvasItem interface, so you can use the
GooCanvasItem functions such as goo_canvas_item_raise() and
goo_canvas_item_rotate().
GooCanvasPath uses the same path specification strings as the Scalable Vector
Graphics (SVG) path element. For details see the SVG_specification.
To create a GooCanvasPath use goo_canvas_path_new().
To get or set the properties of an existing GooCanvasPath, use g_object_get()
and g_object_set().
