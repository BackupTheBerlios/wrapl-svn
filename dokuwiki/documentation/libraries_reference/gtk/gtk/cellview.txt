  A GtkCellView displays a single row of a GtkTreeModel, using cell renderers
  just like GtkTreeView. GtkCellView doesn't support some of the more complex
  features of GtkTreeView, like cell editing and drag and drop.

  Sets the model for cell_view. If cell_view already has a model set, it will

  cell_view : a GtkCellView
  model :     a GtkTreeModel

==== :SetDisplayedRow ====
'':SetDisplayedRow(self @ [[#T|T]], path @ [[..:..:Gtk:Gtk:TreePath#T|Gtk.Gtk.TreePath.T]]) : [[..:..:Std:Object#T|Std.Object.T]]''

  Sets the row of the model that is currently displayed by the GtkCellView. If
  the path is unset, then the contents of the cellview "stick" at their last
  value; this is not normally a desired result, but may be a needed intermediate
  state if say, the model for the GtkCellView becomes temporarily empty.
  cell_view : a GtkCellView
  path :      a GtkTreePath or NULL to unset.
