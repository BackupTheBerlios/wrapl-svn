A GtkRadioToolButton is a GtkToolItem that contains a radio button, that is, a
button that is part of a group of toggle buttons where only one button can be
active at a time.
Use gtk_radio_tool_button_new() to create a new GtkRadioToolButton. use
gtk_radio_tool_button_new_from_widget() to create a new GtkRadioToolButton that
is part of the same group as an existing GtkRadioToolButton. Use
gtk_radio_tool_button_new_from_stock() or
gtk_radio_tool_button_new_from_widget_with_stock() to create a new
GtkRAdioToolButton containing a stock item.
